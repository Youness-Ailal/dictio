noun

Keyboard

/'ki:bc:d/

Meaning

  • The physical keyboard - The actual tactile keyboard used to input data into devices
  • A keyboard that appears on touchscreen devices like smartphones and tablets
  • Special key combinations that allow you to quickly perform commands without navigating menu systems. For example, Ctrl + C to copy and Ctrl + V to paste on Windows machines

Synonym

electronic keyboard

verb

Meaning

  • to type on a computer keyboard
